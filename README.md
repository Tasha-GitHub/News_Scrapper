# News_Scrapper
A web scrapper app that uses mongodb storage, this app allows users to scrape reddit for developer related articles then save an article to the favorites page. It also allows users to take notes on the article. This is saved to each indiviudal note.

# Technologies Used
* Node
* Handlebars
* body-parser
* cheerio
* express
* mongoose
* path 
* request
* html
* css
* bootstrap
